Lumpy Text

lumpy-text is a project that makes you using linear gradient text, such as text color, opacity, text shadow, text size and so on.

Gradient Color

Dolor eaque asperiores repellendus aut modi pariatur tenetur! Repellat mollitia?

$('.lumpy').lumpy({
    color: {
        begin: '#fff',
        end: '#444'
    },
    steps: 60
});

Gradient Direction

Using the property 'direction' to control the direction of gradient text, choosing 'begin' or 'end'. The default value is 'end'. Otherwise, note value of 'color' property, it's using rgb way.

$('.lumpy').lumpy({
    color: {
        begin: [68, 68, 68],
        end: [255, 255, 255]
    },
    steps: 60,
    direction: 'begin'
});

Gradient Opacity

Opacity property something likes 'color', you can set value between 0.00-1.00. Note the 'steps' & 'direction' perperties in opacity object, default 'steps' extends lumpy object, 'direction' property determined where opacity influence in the whole text, and its default value is 'begin'

$('.lumpy').lumpy({
    color: {
        begin: '#ff87a3',
        end: '#00fa64',
        steps: 199,
        direction: 'end'
    },
    opacity: {
        begin: 0.8,
        end: 0.2,
        steps: 100,
        direction: 'end'
    },
    steps: 250,
});

Gradient Size

Font-size makes your text expressive. it likes 'color' & 'opacity' properties. Lumpy supports all the units suffix for 'size' property

$('.lumpy').lumpy({
    color: {
        begin: '#ff87a3',
        end: '#00fa64',
        steps: 199,
        direction: 'end'
    },
    opacity: {
        begin: 0.8,
        end: 0.2,
        steps: 100,
        direction: 'end'
    },
    size: {
        begin: '80px',
        end: '20px',
        steps: 22,
    },
    steps: 300,
});
