Lumpy Text

lumpy-text is a project that makes you using linear gradient text, such as text color, opacity, text shadow, text size...and so on.

Download .zip Download .tar.gz View on GitHub

Gradient Color

Dolor eaque asperiores repellendus aut modi pariatur tenetur! Repellat mollitia?

$('lumpy').lumpy({
    begin: '#fff',
    end: '#444',
    steps: 60
});

Gradient Direction

using the property 'direction' to control the direction of gradient text, choosing 'begin' or 'end'. default is 'end'

$('lumpy').lumpy({
    begin: '#444',
    end: '#fff',
    steps: 60,
    direction: 'begin'
});
